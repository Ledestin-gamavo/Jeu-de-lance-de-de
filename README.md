#  Dice Roller – Desktop App (Python & Tkinter)

Une application desktop interactive développée en **Python avec Tkinter**, simulant un **lancer de dé animé avec effet sonore**, conçue pour illustrer les bases des interfaces graphiques, des événements et de l’expérience utilisateur.

---

##  Fonctionnalités

-  Génération aléatoire d’un nombre entre 1 et 6

-  Animation simulant un vrai lancer de dé

-  Effets sonores synchronisés avec l’animation

-  Interface graphique moderne et épurée

-  Application légère (aucune dépendance externe)


---

##  Technologies utilisées

- **Python 3**

- **Tkinter** (GUI native)

- **random** (génération aléatoire)

- **winsound** (effets sonores – Windows)
